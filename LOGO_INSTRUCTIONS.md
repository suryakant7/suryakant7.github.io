# Company Logo Instructions

To display company/institution logos in your portfolio, add logo image files to the following directory:

```
assets/images/company/
```

## Required Logo Files

### Achievements Section
- `gate-logo.png` - GATE examination logo
- `drdo-logo.png` - DRDO (Defence Research and Development Organisation) logo
- `cdac-logo.png` - C-DAC (Centre for Development of Advanced Computing) logo
- `jio-logo.png` - Reliance Jio logo
- `hackathon-logo.png` - Generic hackathon/trophy icon
- `patent-logo.png` - Patent/innovation icon

### Education Section
- `university-logo.png` - Your university logo
- `cdac-logo.png` - C-DAC logo (reused from achievements)
- `school-logo.png` - School/college logo for Higher Secondary
- `certifications-logo.png` - Generic certification badge

## Image Specifications

- **Format**: PNG with transparent background (preferred) or JPG
- **Size**: Recommended 200x200 pixels (will be displayed at 80x80)
- **Quality**: High resolution for crisp display
- **Aspect Ratio**: Square (1:1) works best

## Where to Find Logos

1. **Official websites**: Download from company's press/media kit
2. **Wikipedia**: Often has official logos in good quality
3. **Create your own**: Use design tools to create simple icons if official logos aren't available
4. **Fallback**: If logo is missing, emoji icons will display automatically

## Example Structure

```
assets/
  images/
    company/
      gate-logo.png
      drdo-logo.png
      cdac-logo.png
      jio-logo.png
      university-logo.png
      school-logo.png

      patent-logo.png
      certifications-logo.png
```

## Notes

- The website includes automatic fallback to emoji icons if logo files are not found
- Logos are styled with drop-shadow effects and hover animations
- All logos should be appropriate for professional display
- Ensure you have rights to use company logos (usually available for non-commercial personal portfolios)
